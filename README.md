# Lofi-and-Chill
## Check out https://lofi-and-chill.vercel.app for a demo!
I built a Lofi, Pomodoro timer website that lets you study and chill, with a function of moving all of the boxes around to your liking. Every day, using NASA's Api to implement their APOD (Astronomy Picture of the Day) onto the page, it gives a cozy vibe, with daily images. With over 10 tracks, I feel like I did a super good job on this project!
